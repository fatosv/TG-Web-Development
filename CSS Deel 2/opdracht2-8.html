De CSS property box-sizing kan gebruikt worden om ervoor te zorgen dat de waarde van width en height gelijk blijft ongeacht de padding die toegevoegd wordt. Bijvoorbeeld: dit is dezelfde box als de vorige opdracht, maar deze is 100px hoog en 100px breed. Dit komt door de box-sizing property die ervoor zorgt dat de hoogte en breedte gelijk blijven aan de originele waarden.
