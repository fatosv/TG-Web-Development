Child selectors

Hiermee selecteer je de directe child element van het eerste element. De syntax is A > B, waarbij B de directe child van A is. Bijvoorbeeld: deze paragraaf is een directe child van de <div> element erboven. Met de child selector kan deze <p> element groen gemaakt worden.

Deze selector is vooral handig als je specifieke child elementen wilt stijlen. Deze <p> element zit niet in de <div> element en wordt dus niet groen.

Adjacent selectors

Deze selector wordt gebruikt voor een element die direct na een andere element komt, zolang beide elementen dezelfde parent element hebben. De syntax is A + B, waarbij B een directe opvolger is van A. Bijvoorbeeld: deze <p> element volgt direct na de <h2> element hierboven en kan dus met de adjacent selector rood gemaakt worden.

Deze <p> element wordt hierbij niet geselecteerd omdat het niet direct volgt na de <h2> element. Deze selector is dus vooral handig als je een specifieke element wilt selecteren zonder dat het andere child elementen meeneemt.

General selectors

In tegenstelling tot de adjacent selector, selecteert de general selector wel alle elementen na het eerste element. De syntax is A ~ B, waarbij B een opvolger is van A en beide elementen dezelfde parent element hebben.

Bijvoorbeeld: met de general selector kunnen alle <p> elementen die volgen na de <h3> element blauw gemaakt worden. Het voordeel hiervan is dat je in een keer meerdere elementen kan bewerken die een specifieke element opvolgen.
