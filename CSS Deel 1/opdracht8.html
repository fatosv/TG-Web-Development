Cascade en Inheritance

Cascade

Cascade gaat over de volgorde waarin de CSS code wordt geschreven. De code wordt namelijk van boven naar beneden gelezen. Dit betekent dat als er twee regels zijn met dezelfde specificiteit, de onderste regel belangrijker wordt gevonden dan de regel erboven. Bijvoorbeeld: als ik voor deze <p> element een regel schrijf dat het blauw moet zijn en daaronder een regel dat het rood moet zijn, dan zal de tekst uiteindelijk rood worden.

Inheritance

Met inheritance wordt bedoeld dat child elementen soms regels van hun parent elementen erfen. Bijvoorbeeld: door deze <div> element groen te maken, worden de child elementen ook groen.

Specificity

Specificiteit is hierin ook belangrijk omdat elementen met een hogere specificiteit als belangrijker worden beschouwd dan elementen met een lagere specificiteit. Bijvoorbeeld: deze <div> element is blauw, maar de <p> element heeft een ID selector en heeft dus een hogere specificiteit. Zo kan ik de <p> element een paarse kleur geven en de inheritance overschrijven.
