Universal selector

Deze selecteert alle elementen. Bijvoorbeeld: met deze selector heb ik alle elementen gecentreerd.

Type selector

Deze selecteert alle elementen met dezelfde naam. Bijvoorbeeld: alle <p> elementen zijn rood.

Class selector

Deze selecteert alleen de elementen die een bepaalde class attribuut hebben gekregen in het html bestand. Bijvoorbeeld: door deze <p> element een class attribuut te geven, kon ik het paars maken in plaats van rood.

ID selector

Deze selecteert alleen het element die een bepaalde ID attribuut heeft gekregen in het html bestand. De ID selector wordt gebruikt voor unieke elementen. Bijvoorbeeld: bovenstaande <h2> element staat rechts in plaats van gecentreerd met behulp van de ID selector.

Descendant selectors

Deze selecteert elementen die in andere parent elementen zitten.

Bijvoorbeeld: deze <p> elementen zitten in dezelfde parent <div> element en kunnen dus apart van de andere <p> elementen bewerkt worden.
