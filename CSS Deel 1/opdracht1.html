Opdracht 1

Er zijn drie manieren om CSS in je HTML in the laden: External, Internal en Inline. External houdt in dat er een aparte CSS bestand wordt gemaakt, hiernaar wordt gelinkt in de head van het HTML bestand. Internal houdt in dat er in de head van het HTML bestand style elementen worden gebruikt om CSS in te schrijven. Inline houdt in dat er binnen een HTML element een style attribuut kan worden toegevoegd.
