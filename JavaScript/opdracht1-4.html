  1. Hier komt 2 en 2 uit. Eerst wordt er een prefix incrementation toegepast, dat betekend dat er 1 bij wordt opgeteld en dat dat de nieuwe value is. Vervolgens wordt er postfix incrementation toegepast. Hier wordt er ook 1 bij opgeteld, maar de originele value blijft hetzelfde. Het blijft dus 2.
  2. Hier komt 1 en 3 uit. Dit komt omdat er eerst een postfix incrementation wordt toegepast waarbij de value hetzelfde blijft maar er wel 1 bij wordt opgeteld. Bij de prefix implementation wordt daar vervolgens weer 1 bij opgeteld.
  3. Hier komt 4 en 2.4 uit. 10 past 2 keer in 24 en wat er dan overblijft (de modulo) is 4. 24 gedeeld door 10 is 2.4
  4. Hier komt false uit. De logical NOT operator veranderd true naar false en vice versa. In dit geval is het dus true, maar wordt het door de ! als false verklaard.
    1. Hier komt false uit omdat het eigenlijk dus true moet zijn.
    2. Hier komt false uit, het is inderdaad een Boolean en wordt dus als false gezien door deze operator.
    3. Hier komt true uit omdat dat het tegenovergestelde is van false.
    4. Hier komt false uit omdat het een string is die eigenlijk true is.
    5. Hier komt false uit omdat dat het tegenovergestelde is van true.
    6. Hier komt false uit omdat het een string is die eigenlijk true is.
    7. Hier komt true uit omdat het eigenlijk false is dat de waarde null is.
    8. Hier komt true uit omdat het eigenlijk false is dat de waarde undefined is.
  6. Met addition assignment wordt er een waarde toegevoegd aan een variabale. Dus x += y is gelijk aan x = x + y. Dit werkt ook met strings, dan wordt bijvoorbeeld tekst y aan tekst x toegevoegd.
  7. Met subtraction assignment wordt er een waarde afgehaald van een variabele. Dus x -= y is gelijk aan x = x - y. Hier kunnen geen strings gebruikt worden.
  8. Met multiplication assignment wordt een variabele vermenigvuldigd. Dus x *= y is gelijk aan x = x * y. Hier kunnen geen strings gebruikt worden.
  9. Met division assignment wordt een variabele gedeeld. Dus x /= y is gelijk aan x = x / y. Hier kunnen geen strings gebruikt worden.
  10. Met remainder assignment wordt een remainder toegekend aan een variabele. Dus x %= y is gelijk aan x = x % y. Hier kunnen geen strings gebruikt worden.
  11. Met exponentiation assignment wordt een variabele verhoogd met een bepaalde exponent. Dus x **= y is gelijk aan x = x ** y. Hier kunnen geen strings gebruikt worden.
