Position Values

  • Static position: dit is de standaard position value van een html document. Het element is gepositioneerd volgens de normale flow van het document.
  • Absolute position: het element valt buiten de normale flow van het document en kan gepositioneerd worden met de top, right, bottom en left waardes. Het element wordt gepositioneerd relatief tov de body en andere elementen zullen het negeren.
  • Relative position: het element is gepositioneerd volgens de normale flow van het document. Het kan gepositioneerd worden met de top, right, bottom en left waardes relatief tov zichzelf.
  • Fixed position: het element valt buiten de normale flow van het document en andere elementen zullen het negeren. Het kan gepositioneerd worden relatief tov van de body en blijft zichtbaar wanneer er wordt gescrold.
  • Sticky position: het element is gepositioneerd volgens de normale flow van het document en kan gepositioneerd worden relatief tov zichzelf. Zodra er wordt gescrold en de top waarde is bereikt, gedraagd het element zich hetzelfde als fixed position.
