Pseudo Elementen

  • ::after: met deze pseudo element kan er content na het geselecteerde element geplaatst worden.
  • ::before: met deze pseudo element kan er content voor het geselecteerde element geplaatst worden.
  • ::first-letter: met deze pseudo element kan de eerste letter van de eerste regel van het geselecteerde element bewerkt worden.
  • ::first-line: met deze pseudo element kan de eerste regel van het geselecteerde element bewerkt worden.
  • ::file-selector-button: met deze pseudo element kan de button bewerken van een <input> element met type="file".
  • ::backdrop: met deze pseudo element kan de achtergrond bewerkt worden van een element dat in full-screen mode bekeken wordt.
  • ::grammar-error: met deze pseudo element kan een stuk tekst worden bewerkt dat grammaticale fouten bevat (bepaald door de user agent).
  • ::spelling-error: met deze pseudo element kan een stuk tekst worden bewerkt dat spellingsfouten bevat (bepaald door de user agent)
  • ::selection: met deze pseudo element kan een met de muis geselecteerde stuk van het element bewerkt worden.
  • ::marker: met deze pseudo element kunnen de markers van een lijst bewerkt worden.
  • ::target-text: met deze pseudo element kan een stuk tekst worden geselecteerd en bewerkt waar de gebruiker heen was gescrold.
  • ::part: met deze pseudo element kan het part attribuut in een shadow tree worden bewerkt.
  • ::placeholder: met deze pseudo element kan de tijdelijke placeholder tekst in een <input> element bewerkt worden.
  • ::slotted: met deze pseudo element kan een <slot> element in een template bewerkt worden.
